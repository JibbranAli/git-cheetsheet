# Git History Commands

## git log
\\\ash
git log --oneline --graph --decorate --all
\\\

## git show
Show specific commit details.
\\\ash
git show <commit>
\\\

## git blame
See who changed each line of a file.

## git bisect
Locate bugs by binary search.
\\\ash
git bisect start
git bisect bad
git bisect good <good-commit>
\\\
