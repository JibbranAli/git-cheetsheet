# Basic Git Commands

## git init
Initialize a new repository.
\\\ash
git init
\\\

## git add
Stage files for commit.
\\\ash
git add .
\\\

## git commit
Save a snapshot of staged changes.
\\\ash
git commit -m "Initial commit"
\\\

## git status
View status of working directory.
\\\ash
git status
\\\

## git diff
Show changes between commits or working directory.
\\\ash
git diff
\\\
