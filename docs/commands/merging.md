# Merging in Git

## Fast-Forward Merge
\\\ash
git merge feature-login
\\\

## Merge Commit
\\\ash
git merge feature-login --no-ff
\\\

## Squash Merge
\\\ash
git merge --squash feature-login
\\\

## Handling Merge Conflicts
Fix conflict markers then run:
\\\ash
git add .
git commit
\\\
