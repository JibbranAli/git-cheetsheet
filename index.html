Basic Commands

Essential commands to get started

5 Commands
Essential

git init

Initialize a new Git repository in your project directory

Command 
git init
Run this in your project folder to start version control
Essential

git add

Stage files for the next commit

Command 
git add .                    # Add all files
git add file.txt            # Add specific file
git add src/                # Add directory
Stage changes before committing to selectively include files
Essential

git commit

Save a snapshot of your staged changes

Command 
git commit -m "Your message"
git commit -am "Quick commit"  # Add & commit
Write clear, descriptive commit messages
Essential

git status

Check the state of your working directory

Command 
git status
git status -s              # Short format
Use this to see what files are modified, staged, or untracked
Useful

git diff

View differences in your code

Command 
git diff                    # Unstaged changes
git diff --staged           # Staged changes
Review changes before committing

Branching

Create isolated work environments

3 Commands
Essential

git branch

List or create branches

Command 
git branch                  # List branches
git branch feature-name    # Create branch
git branch -d branch-name  # Delete branch
Branches let you work on features without affecting main code
Essential

git switch

Switch between branches

Command 
git switch main
git switch -c new-branch   # Create & switch
Modern way to change branches (replaces git checkout)
Legacy

git checkout

Traditional way to switch branches

Command 
git checkout main
git checkout -b new-branch
Still widely used, but git switch is preferred

Merging

Combine changes from branches

2 Commands
Essential

git merge

Combine branches into one

Command 
git merge feature-branch
git merge --no-ff feature   # Force merge commit
git merge --squash feature  # Squash commits
Integrate feature work back into main branch
Important

Merge Conflicts

Resolve conflicts when Git can't auto-merge

Command 
# Edit conflicted files
git add .
git commit
Git marks conflicts with special markers - edit and resolve

History & Inspection

Explore commit history

2 Commands
Essential

git log

View commit history

Command 
git log
git log --oneline
git log --graph --all
git log -5                # Last 5 commits
See the evolution of your project
Useful

git show

View specific commit details

Command 
git show
git show commit-hash
git show HEAD~1
See exactly what changed in a commit

Undoing Work

Safely undo changes

3 Commands
Caution

git reset

Move branch pointer backward

Command 
git reset --soft HEAD~1   # Keep changes staged
git reset --mixed HEAD~1  # Keep changes unstaged
git reset --hard HEAD~1   # Discard all ⚠️
Use carefully - can lose work!
Safe

git revert

Safely undo commits

Command 
git revert HEAD
git revert commit-hash
Creates new commit - safe for shared repos
Essential

git stash

Temporarily save changes

Command 
git stash
git stash pop              # Apply & remove
git stash list
Save work in progress to switch branches

Remote Repositories

Work with GitHub, GitLab, etc.

4 Commands
Essential

git remote

Manage remote connections

Command 
git remote -v              # List remotes
git remote add origin URL  # Add remote
Connect your local repo to GitHub
Essential

git push

Upload commits to remote

Command 
git push origin main
git push -u origin main    # First time
Share your work with others
Essential

git pull

Download and merge changes

Command 
git pull
git pull origin main
Get latest changes from remote
Essential

git clone

Copy repository to local

Command 
git clone URL
git clone URL folder-name
Get started with existing projects
